📖 MyBiOut! 手册

概述

MyBiOut!（My-Bilibili-Output，导出我的哔哩哔哩）是一个综合性的、一站式开箱即用的哔哩哔哩导出工具集。

使用 FastAPI 构建，网页前端部署在 localhost。系统包含三大功能模块：本地缓存导出、BBDown 可视化封装、Markdown 格式化导出。

当前仅支持 Windows 环境运行

安装与启动

安装

通过 PyPI 安装（需要 Python 3.14+ 环境）：

pip install mybiout

启动服务

使用以下命令启动（不区分大小写）：

MyBiOut!

默认在 23333 端口启动服务。访问 http://localhost:23333 打开界面。

自定义端口

MyBiOut! --port 端口号

例如：MyBiOut! --port 2026

启动动画

启动时会播放一段终端动画，包含直升机飞行、盲文粒子特效、随机配色主题等。终端尺寸不足时自动降级为静态 Banner。

功能模块

🎞️ LocalOut! — 本地缓存导出

导出本地的哔哩哔哩视频缓存，支持以下来源：

  • PC 桌面端缓存：自动解析 videoInfo.json，合并 .m4s 文件
  • Android 设备缓存（ADB）：通过 USB 连接扫描设备上的缓存，解析 entry.json
  • 自定义本地路径：浏览选择任意包含缓存的目录

操作流程：添加源 → 扫描 → 选择视频 → 添加到任务 → 导出

支持自定义导出文件名（BV号、标题、UP主、分P号、时间等任意组合）、并发导出控制、不完整标题降级策略等。

📥 BBDown! — 视频下载

对知名命令行工具 BBDown 的可视化封装。在网页界面上输入视频链接即可下载。

支持的输入格式：

  • BV号、av号、ep号、ss号
  • 完整视频链接
  • b23 短链

可配置项包括：API 模式（默认/TV/APP/国际版）、画质偏好、编码偏好、下载内容（完整视频/仅音频/仅弹幕等）、分P选择、弹幕下载、字幕跳过等。

下载任务自动排队，支持取消、重试、清空操作。

📝 MdOut! — Markdown 导出

将 B 站内容导出为格式化的 Markdown 文档，支持三种类型：

  • 视频信息：包含 UP 主、数据统计、标签、分P列表等
  • 用户信息：包含用户资料、数据统计、收藏夹内容
  • 专栏文章：包含文章元信息、数据统计

内置 Markdown 预览，支持 Rich/Raw 切换查看。导出前可实时预览最终效果。

⚙ 设置（OhMyConfig）

通过首页「哦，我的设置」链接进入设置页面。所有设置分为四组：

  • 通用设置：导出根目录（默认 C:\MyBiOut!）、大模型 API 配置
  • LocalOut 设置：输出目录名、Android 扫描开关、PC 缓存路径、文件命名规则、FFmpeg 并发数等
  • BBDown 设置：Cookie、画质/编码偏好、文件名模板、弹幕/字幕/封面选项等
  • MdOut 设置：SESSDATA、导出选项（封面/标签/统计）、收藏夹详情程度、请求间隔等

涉及敏感信息（API Key、Cookie、SESSDATA）的设置会弹出明文存储风险提示。

设置以 config.ini 存储在项目目录下。

Man! 手册页

就是你现在正在看的这个页面。

左侧是传统手册，右侧是「What can I say about...」AI 对话助手。

对话助手使用设置中配置的大模型 API，并通过 pmc 工具打包项目完整源代码作为上下文。如果未配置 API 或调用失败，将降级为内置的「狗屁不通文章生成器」——产生一段与你的问题主题相关但毫无意义的文字。

点击「直接说」按钮可跳过 API 调用，直接使用狗屁不通文章生成器。
